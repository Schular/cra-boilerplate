This project is a boilerplate over [Create React App](https://github.com/facebook/create-react-app).

## Boilerplate containing
- Axios
- Express server
- PropTypes
- React-Redux w/ Thunk & Reselect
- Styled-Components
- Eslint
- Absolute imports
- Jest
